大溫暖社會福利套案之旗艦計畫
新長期失能人口照護十年計畫

建構完整之我國長期照顧體系，保障身心功能障礙者能獲得適切的服務， 增進獨立生活能力，提升生活品質，以維持尊嚴與自主。

全面性的照護

以全人照顧、在地老化、多元連續服務為長期照顧服務原則，加強照顧服務的發展與普及。

多元的服務方案

保障民眾獲得符合個人需求的長期照顧服務，並增進民眾選擇服務的權利。

家庭照護協助

支持家庭照顧能力，分擔家庭照顧責任。

完整的照護支援機制

建立照顧管理機制，整合各類服務與資源，確保服務提供的效率與效益。

政府經費補助

透過政府的經費補助，以提升民眾使用長期照顧服務的可負擔性。

永續經營

確保長期照顧財源的永續維持，政府與民眾共同分擔財務責任。

照顧服務培訓課程

因應我國長期照護人力需求，提昇照顧服務品質，促進居家服務員、病患服務人員就業市場相互流通，增加就業機會，並整合居家服務員、病患服務人員訓練課程為照顧服務員訓練課程

主管機關、服務對象及服務項目

1.主管機關:中央為內政部、行政院衛生署，在地方為直轄市、縣（市）政府。
2.服務對象：日常生活活動功能或維持獨立自主生活能力不足，需他人協助者。
3.服務項目：
3.1 生活照顧服務。
3.2 身體照顧服務。
3.3 在護理人員指導下執行病患照顧之輔助服務。但服務範疇不得涉及醫療及護理行為。

受訓對象、單位

1.受訓對象:年滿十六歲以上、身體健康狀況良好，具擔任照顧服務工作熱忱者。
2.訓練單位：接受直轄市、縣（市）政府委託辦理本計畫者，或符合下列資格之單位且具合格實習訓練場所，或與合格實習訓練場所定有合作計畫者，得擬具計畫，以核心課程訓練地之所在為準，送當地直轄市、縣（市）政府審查核定：
3.服務項目：
3.1 生活照顧服務。
3.2 身體照顧服務。
3.3 在護理人員指導下執行病患照顧之輔助服務。但服務範疇不得涉及醫療及護理行為。

Learn more

實習訓練場所

1 經直轄市、縣（市）政府督導考核成績優良之醫院。
2 經行政院衛生署或直轄市、縣（市）政府督導考核成績優良之護理機構。
3 經內政部或直轄市、縣（市）政府評鑑甲等以上之公立或財團法人老人長期照顧機構、身心障礙住宿機構、居家服務提供單位、日間照顧服務提供單位。

Learn more

師資條件

1.與授課主題相關之大專院校醫學、護理學、營養學、法律或社會工作相關科系所講師以上資格者。
2.與授課主題相關之大學以上畢業，且具實務工作經驗三年以上者。
3.與授課主題相關之實務經驗五年以上者（限實習、照顧技巧實務課程）。

Learn more

成績考核

1 受訓對象參加核心課程之出席率應達百分之八十以上，並完成所有回覆示教課程及臨床實習課程者，始可參加成績考核。
2 結業證明：
2.1 訓練期滿後，訓練單位應將結訓人員名冊、出席情形及考核成績等相關資料，以核心課程訓練地之所在為準，送當地直轄市、縣（市）政府備查。
2.2 經考評及格者，由訓練單位核發結業證明書；訓練單位並應將所在地直轄市、縣（市）政府同意備查之日期、文號載明於結業證明書內，以利查核。

Learn more

訓練課程內容與時數：

1.核心課程：五十小時。
2.實習課程：四十小時（含回覆示教十小時、臨床實習三十小時）。
3.直轄市、縣（市）政府得依其業務需要增列照顧服務員分科訓練課程內容與時數。

社區照顧關懷據點

"由有意願的村里辦公處及民間團體參與設置，邀請當地民眾擔任志工，提供老人關懷訪視、電話問安諮詢及轉介服務，並視當地需求特性，提供餐飲服務或辦理健康促進活動，每個據點均可提供3 項以上的服務"

社區照顧關懷據點特色

多種健康活動

透過設置社區照顧關懷據點可以讓老人從家裡走到關懷據點參加精心設計安排的健康促進活動。

維持社交、認識人群

可認識社區中其他的老人與熱心的志工，大家閒話家常、分享生活點滴，並可使用據點內的健康器材等，達到身心健康的效果。

主動探訪、主動關懷

另針對平日較少出門或失能的長者，關懷據點亦可以主動出擊方式，提供社區老人關懷訪視、電話問安、諮詢及轉介等服務。

自助互助、永續經營

建立互助與溫暖的居住環境，是我們的共同目標，讓老人能在熟悉的環境，與親人、鄰居、老朋友一起而做做活動、互相關心、分享生命的經驗，是我們積極想營造的共生關係及利於老人居住的健康環境。
